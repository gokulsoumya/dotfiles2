# Text

## Manipulation

- `cut` - split into columns
  - `cut -d ' ' -f 1` - select first space delimited column
  

## Stats

- `wc` - count words, lines, etc
- `grep` - search text in file
  - `grep pattern file.ext file2.ext...`
